# КУБ (KybVision) — MVP платформа проектных инициатив

КУБ — это MVP-платформа, которая помогает пройти путь от идеи до MVP:
- определить трек;
- оформить карточку проекта;
- собрать команду;
- подобрать возможности (гранты, конкурсы, хакатоны и т.д.);
- получить подсказки от AI-наставника.

## Быстрый запуск

1. Установите `Node.js` и `npm`.
2. Установите зависимости:

```bash
npm install
```

3. Запустите проект:

```bash
npm run dev
```

4. Откройте в браузере:

`http://localhost:3000`

## Основные разделы

- `/` — landing
- `/dashboard` — дашборд
- `/project` — карточка проекта
- `/team` — команда
- `/opportunities` — возможности
- `/ai` — AI-наставник
- `/profile` — профиль
- `/admin` — demo-админка
